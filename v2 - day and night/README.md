# Version 2 — Day and Night

A lightweight and clean wallpaper simulation that displays a floating year centerpiece against a realistic sky background. The sky cycles dynamically between day and night based on the local system clock.

![Status](https://img.shields.io/badge/Status-Deprecated-red.svg)


## 🌟 Features

* **Dynamic Sky Cycles**: Automatic transitions between Day (sky blue/white gradient) and Night (deep grey/black gradient) based on the local system hour.
* **Celestial Movements**:
  - A bright **Sun** rising and setting along a parabolic path during daylight hours.
  - A realistic **Moon** traveling across the night sky, with its horizontal position dynamically shifting depending on the day of the month.
* **Canvas Twinkling Stars**: A layer of twinkling stars that automatically fades in during the night phase.
* **Shooting Stars**: Random shooting stars that cross the sky during the night hours.
* **Floating Centerpiece**: The current year displayed in a floating animation with retro-style offset drop shadows.
* **Date Stamp**: A digital date indicator located in the top-left corner.


## 🚀 How to Run

This version runs entirely client-side without any build dependencies or local server requirements. 

Simply open the `index.html` file in any modern web browser:
```bash
# Double-click index.html or open via command line:
open index.html
```
