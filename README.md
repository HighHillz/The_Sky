# The Sky — Dynamic Wallpaper

A beautiful, dynamic, and interactive live wallpaper simulation built with HTML5, CSS3, and Vanilla JavaScript. The wallpaper shifts through the cycles of the day (Sunrise, Day, Sunset, Night) and adapts its elements to match the real-time system clock, astronomical movements, and live weather conditions.
